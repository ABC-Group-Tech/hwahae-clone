Build the TERMS OF USE page for a cosmetics review platform called "ABCPharm".

## Important Rules
- Do NOT generate AI images except for the logo
- Use placeholder boxes for all image areas (bg-gray-200 + text label)
- ALL UI TEXT MUST BE IN KOREAN (한글)

## Tech Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Shadcn/ui components
- Lucide React icons

## Brand Colors
- Primary color: #f39800 (orange)
- Secondary text: #727171 (gray)
- Background: #FFFFFF (white)
- Border: #E5E5E5

## Overall Layout
- Mobile-first design (max-width: 480px, centered)
- Font: Pretendard or system-ui

# TERMS OF USE PAGE (이용약관)

## Layout

### Header
- Close button (X) - left side
- Title (Korean): "이용약관" - centered
- Background: white

### Table of Contents
- Header (Korean): "< 목차 >"
- Numbered list of sections (clickable links):
  1. ABCPharm 이용약관
  2. ABCPharm 위치기반서비스 이용약관

### Terms Content Container
- Full width
- Scrollable content area
- Rendered in iframe or native scroll container
- Background: white
- Padding: 16px
