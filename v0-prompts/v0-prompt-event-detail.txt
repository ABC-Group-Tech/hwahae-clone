# EVENT DETAIL PAGE (이벤트 상세)

## Layout

### Header
- Back button (←)
- Share button (Share2 icon)
- Background: transparent or white

### Event Banner Section
- Large banner image: placeholder (full width, aspect-ratio: 16:9 or 4:3)
- May contain multiple banners (carousel)
- Rounded corners: 0 or 12px
- Promotional graphics placeholder

### Event Info Section
- Event period badge:
  - Icon: Calendar
  - Text (Korean): "[시작일] ~ [종료일]"
  - Format: "2024.01.15 ~ 2024.02.15"
- Event status badge (if applicable):
  - "진행중" (green)
  - "종료임박" (orange)
  - "종료" (gray)
- Event title (Korean): Large, bold
- Event description: Markdown or plain text

### Event Benefits Section (if applicable)
- Header (Korean): "이벤트 혜택"
- Benefit cards:
  - Icon + text
  - Examples: "무료배송", "추가할인", "사은품 증정"

### Event Products Section
- Header (Korean): "이벤트 상품"
- Product count: "[N]개 상품"

#### Product Grid (2 columns)
- Product card:
  - Event badge (top-left):
    - Sale percentage: "50% OFF" (red/orange background)
    - Special label: "특가", "한정수량"
  - Product image: placeholder (square, rounded)
  - Brand name (gray, small)
  - Product name (black, 2 lines max)
  - Price section:
    - Original price: strikethrough, gray
    - Event price: orange (#f39800), bold
  - "품절" badge (if sold out):
    - Gray overlay
    - Text: "품절"
  - Add to cart quick button (optional)

### Multiple Event Sections
- If event has multiple product groups:
  - Section headers with descriptions
  - Collapsible sections

### Event Terms Section
- Collapsible/Accordion
- Header (Korean): "유의사항" or "이벤트 안내"
- Expand icon (ChevronDown)
- Content:
  - Numbered or bulleted list
  - Terms and conditions text
  - Refund/exchange policy
  - Event restrictions

### Related Events Section (optional)
- Header (Korean): "다른 이벤트"
- Horizontal scroll of event cards
- Event card:
  - Thumbnail placeholder
  - Event title
  - Period

### Footer
- Same as other pages

## INTERACTIONS

- Banner carousel: Swipe or auto-play
- Product card: Navigate to product detail
- Add to cart: Add product to cart (with quantity selector)
- "품절" items: Disabled, can't add to cart
- Terms accordion: Expand/collapse
- Share button: Share event link
